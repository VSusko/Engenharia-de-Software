\doxysubsection{Detailed Description}
Represents a system model consisting of systems and flows. 

The \mbox{\hyperlink{classModel}{Model}} class defines a model that holds systems and flows. It provides methods to manage the components (add, remove) and perform operations such as simulation and reporting.
